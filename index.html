Red

Clasificación automática de imágenes mediante redes neuronales convolucionales (CNN). Este proyecto demuestra cómo un modelo entrenado es capaz de distinguir entre perros y gatos usando TensorFlow.js.

PROBAR MODELO

¿Qué Hace el Modelo?

Nuestro modelo CNN analiza imágenes y predice si se trata de un perro o un gato usando filtrados, pooling y capas densas.

Red Convolucional

Utiliza filtros que extraen características como bordes, formas y patrones importantes dentro de la imagen.

MaxPooling

Reduce la dimensionalidad de las imágenes manteniendo información importante, aumentando la eficiencia del modelo.

Clasificación

El modelo final predice la probabilidad de que la imagen sea de un gato o de un perro utilizando una capa de salida con dos neuronas.

Sobre Nuestro Proyecto

Este proyecto fue desarrollado como trabajo final para la materia de Redes Neuronales. Consiste en el entrenamiento e implementación de un modelo de redes neuronales convolucionales exportado a TensorFlow.js para ser ejecutado directamente en el navegador.
